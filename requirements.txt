
### `requirements.txt`
The `requirements.txt` file contains a list of Python dependencies needed for ITMfinder to work. This allows users to install all necessary packages easily. Below is a basic example:

#### Example `requirements.txt` Content:
biopython==1.81
gunicorn==18.0
logan==0.5.9.1
numpy==1.21.6
pandas==1.3.5
python==3.6
